
# AGU_BOOK_LIBRARY
AGU Book Library is a full-stack web application that allows university students to list, rent, and manage books within a secure digital marketplace.
The system provides role-based access control and JWT authentication  to ensure secure transactions and user management.

**Frontend:** React.js
**Backend:** ASP.NET Core Web API (.NET)
**Database:** SQL Server

# The photo below are the project website pages screenshot.


# System Architecture
•	React.js frontend communicating with ASP.NET Core Web API.
•	RESTful API structure for managing users and book resources.
•	JWT-based authentication and authorization.
•	SQL Server relational database for structured data storage.

# Technologies Used:
•	**React.js:** Frontend framework for building responsive and dynamic user interfaces.
•	**ASP.NET Core Web API:** Backend framework for building secure RESTful services.
•	**Entity Framework Core:** ORM used for database operations and schema management.
•	**SQL Local Server:** Relational database management system.
•	**JWT Authentication:** Secure user authentication and authorization mechanism.
•	**RESTful Routing:** Structured API design supporting full CRUD operations.
